## Problem

The program mentions optional bonuses: **TDD**, **junior-led demo**, **peer review**, **creative** engineering notes.

## Value

Extra scoring bands where the team chooses to invest.

## Ideas (pick what fits)

- [ ] **TDD (+5):** One small feature with visible red → green → refactor commits.
- [ ] **Junior-led demo (+5):** Rotate presenter; rehearse timing.
- [ ] **Peer review (+5):** Document a constructive review on another fork.
- [ ] **Creative (+10):** e.g. structured error codes on bus outcomes, OTEL-ready logging — document intent in README.
